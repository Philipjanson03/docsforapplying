## üéì Universities with Officially Stated Minimum Average

### Only One University Officially States Minimum Average:

**[[Docs for applying/Politecnico di Milano\|Politecnico di Milano]]**
- **Minimum Average for Iran:** **15.5/20** (officially stated)
- Source: PoliMi official admissions website
- **Note:** This is the ONLY university that officially publishes minimum GPA requirements for Iranian students

### Universities Without Official Minimum Average:
All other universities evaluate applications holistically without officially stating minimum GPA requirements. However, competitive applicants typically have:
- **14+/20** (70%+) - Minimum competitive
- **15+/20** (75%+) - Strong candidate
- **16+/20** (80%+) - Top candidate
