## ‚úÖ Pre-Application Checklist

### 3-6 Months Before Deadlines:
- [ ] Research universities and programs
- [ ] Take IELTS/TOEFL exam (need scores before deadlines)
- [ ] Take GRE exam (if applying to [[Docs for applying/Netherlands/TU Delft\|TU Delft]])
- [ ] Request official transcripts from university
- [ ] Identify professors for recommendation letters
- [ ] Start drafting motivation letter
- [ ] Update CV to European format

### 1-3 Months Before Deadlines:
- [ ] Get documents officially translated (if needed)
- [ ] Finalize motivation letter
- [ ] Request recommendation letters
- [ ] Prepare portfolio (if required)
- [ ] Gather financial documents
- [ ] Create accounts on university application portals

### 1 Month Before Deadlines:
- [ ] Complete all application forms
- [ ] Upload all required documents
- [ ] Pay application fees
- [ ] Submit applications (submit early, not on deadline day!)
- [ ] Keep copies of all submitted documents
